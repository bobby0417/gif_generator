# Animated GIF Generator

## Description

This project converts multiple images into an animated GIF using Python.

## Features

- Select multiple images
- Image preview
- Automatic resizing
- Custom animation speed
- Save GIF
- Error handling
- Simple Tkinter GUI

## Requirements

- Python 3.10+
- Pillow

## Install

```bash
pip install -r requirements.txt
```

## Run

```bash
python gif_generator.py
```
